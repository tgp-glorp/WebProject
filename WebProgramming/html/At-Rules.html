At-Rules

What are At-Rules:

At rules are CSS-Statements which tell CSS how to behave, they are structured in the following order:
@Identifier(Rule);

Types Of At-Rules:

There are 2 types of @Rules , the first is called the Statement-@Rules and is of the form shown above. The second type of @Rules are the Block-@Rules which are of the following form:
@identifier(Rule){
}
Unlike the Statement-@Rules, Block-@Rules are considered to end at the }. The inside of the block (between the brackets) can contain various things ranging from property declarations to other @Rules.

Examples:

@import:

@import is one of the more commonly used @rules and it tells CSS to import an external style sheet such as typography.css, some examples of this are:
@import url(“typography.css”);
@import url(“layout.css”);
The typography.css style sheet contains some specifications some of which are: 

      body { 
      font-family: 'Helvetica, Arial, sans-serif';
       font-size: 16px; 
      line-height: 1.5; 
      color: #333; 
      }
      /* Headings */
       h1 { 
      font-size: 2.5em;
       font-weight: bold; 
      margin-bottom: 0.5em; 
      }
Similarly, the layout style sheet has its own set of specifications and using the @import allows for easy access to these completed style sheets without requiring effort.

@property:

@property is among the more unique @rules since it allows for the user to create a new property, this property can choose to inherit from other properties and this property can have its own default values, its own type checking , an example: 

      @property --main-color {
          syntax: "";
          inherits: true;
          initial-value: red;
      }
      
      /* Using the custom property */
      body {
          background-color: var(--main-color);
      }
      
      h1 {
          color: var(--main-color);
      }
      
      button {
          background-color: var(--main-color);
          color: white;
      }
This new custom property called --main-color expects a color value, its default value is red which is inherited from the original color class and it applies to the body, h1 and button tags.
